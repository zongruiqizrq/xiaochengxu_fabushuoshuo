# 微信小程序——发布说说
## 前台
### 微信小程序 
#### 项目功能
- 可以发布心情
- 可以上传图片
- 上传的图片、已经发布的心情内图片及用户头像都可与预览
- 在同一网络环境下 都可以发布（必须添加权限，在项目内添加项目成员）
#### 后台介绍
- 电脑中有MongoDB（数据库）
- npm install 安装依赖
#### 运行项目
- mongod --dapath 自己的数据库文件夹
- 开启后端 node app.js
- 进入小程序就可以运行


## 后台
